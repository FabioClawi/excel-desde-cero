¿Qué es Excel?

Excel es una aplicación de hojas de cálculo de la suite Microsoft 365 que permite organizar, analizar y presentar datos de forma eficiente. Es ampliamente utilizada en entornos profesionales y académicos para listas, presupuestos, reportes y análisis básicos.

Un libro es el archivo que contiene una o varias hojas. Cada hoja está compuesta por una cuadrícula de celdas. A su vez, un conjunto contiguo de celdas forma un rango.

Consejo: Piensa en un libro como un archivador, en cada hoja como una sección y en las celdas como las casillas donde anotas la información.

Acceso a Excel

Excel está disponible en diversas plataformas. Las versiones de escritorio para Windows y macOS se instalan localmente y ofrecen el conjunto completo de funcionalidades. La versión web de Microsoft 365 permite trabajar en línea desde cualquier navegador moderno. Esta versión se sincroniza con OneDrive y facilita la colaboración en tiempo real.

Nota: Para usar Excel en la web necesitas una cuenta de Microsoft. Los documentos se guardan automáticamente en OneDrive.

Interfaz de Excel

La interfaz de Excel se organiza en secciones claramente diferenciadas:

  • Barra de título: muestra el nombre del libro y de la aplicación.
  • Cinta de opciones: agrupa los comandos en fichas o pestañas como Inicio, Insertar, Disposición de página, Fórmulas, Datos, Revisar y Vista.
  • Barra de fórmulas: permite ver y editar el contenido de la celda activa.
  • Área de trabajo: la rejilla donde se encuentran las celdas.
  • Barra de estado: ofrece información contextual como el modo de vista, el zoom y cálculos rápidos de las celdas seleccionadas.

Elementos básicos de la hoja

La hoja de cálculo se estructura con columnas identificadas por letras (A, B, C, …) y filas identificadas por números (1, 2, 3, …). Cada intersección forma una celda, que se referencia mediante su columna y fila (por ejemplo, A1, B3).

Los rangos son conjuntos de celdas; pueden ser contiguos (A1:C3) o no contiguos (A1,A3,B2).

Importante: La celda activa es la que está enmarcada y lista para recibir datos. Su referencia aparece en el cuadro de nombres situado junto a la barra de fórmulas.

Tipos de datos y formatos básicos

Excel admite distintos tipos de datos:

  • Texto: cadenas de caracteres. Se alinean a la izquierda por defecto.
  • Número: valores numéricos. Se alinean a la derecha por defecto.
  • Fecha/Hora: representan fechas y horas; se pueden formatear de múltiples maneras.
  • Moneda: números con símbolo monetario y separador de miles.
  • Porcentaje: números interpretados como fracción de cien.

El formato de celda determina cómo se muestra el valor, sin cambiar el valor subyacente.

Fórmulas y referencias

Las fórmulas comienzan con el signo = y utilizan funciones para calcular resultados. Por ejemplo: =SUMA(A1:A10) suma los valores del rango A1 a A10.

Las referencias relativas (A1) cambian al copiar la fórmula a otra celda. Las referencias absolutas ($A$1) se mantienen fijas. Las referencias mixtas combinan ambos comportamientos ($A1 o A$1).

Nota: Si una fórmula produce un error (#¡DIV/0!, #N/A, #¡VALOR!, etc.), revisa los argumentos y tipos de datos utilizados.

Fichas clave de la cinta

Excel organiza los comandos en pestañas. Estas son las principales fichas:

  • Inicio: portapapeles, formato de fuente, alineación, número, estilos de celda y edición.
  • Insertar: tablas, gráficos, ilustraciones, texto y símbolos.
  • Disposición de página: configuración de márgenes, orientación, tamaño de papel y área de impresión.
  • Fórmulas: biblioteca de funciones, creación de nombres definidos y auditoría de fórmulas.
  • Datos: herramientas para obtener, ordenar, filtrar y validar datos.
  • Revisar: ortografía, comentarios y protección de hojas/libros.
  • Vista: opciones de zoom, vistas personalizadas y congelar paneles.

Guardar y tipos de archivo

Al guardar tu libro, puedes elegir diferentes formatos según tus necesidades:

  • .xlsx: formato estándar para hojas de cálculo.
  • .xlsm: permite almacenar macros.
  • .csv: datos separados por comas, útil para intercambio con otras aplicaciones.

Utiliza nombres descriptivos y guarda con frecuencia para no perder tu trabajo.
